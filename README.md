Project Name: NGO Volunteer Management System

Description:
This project is a simple command-line based system to manage NGO volunteers.
It allows adding, viewing, searching, deleting, and counting volunteers.

Features:
- Add Volunteer
- View All Volunteers
- Search Volunteer
- Delete Volunteer
- Count Total Volunteers

Technology Used:
- Python
- CSV file handling

How to Run:
1. Open terminal
2. Run: python ngo_project.py

Purpose:
This project helps NGOs manage volunteer data efficiently and reduce manual work.
